# 三维辐射传输方程物理信息神经网络：研究总结与论文写作指南

## 摘要

本文档系统总结了基于物理信息神经网络（PINN）求解三维辐射传输方程（RTE）的研究工作，特别是针对各向异性散射问题的研究。我们开发了一套完整的计算框架，包括：(1) 高保真前向蒙特卡洛（FMC）求解器用于生成基准解，(2) 基于RMC的验证套件用于交叉验证，(3) 物理修正的PINN训练与验证流程，以及(4) 符合学术出版标准的分析工具。该框架针对三个典型测试案例进行了验证：纯吸收案例（Case A）、各向同性散射案例（Case B）和强前向各向异性散射案例（Case C）。

---

## 1. 引言与研究动机

### 1.1 问题陈述

三维直角坐标系中的稳态辐射传输方程：

```
(s·∇)I(r,s) + β(r)I(r,s) = κ(r)I_b(r) + σ_s(r)/(4π) ∫ I(r,s')Φ(s,s')dΩ'
```

其中：
- `I(r,s)`：位置 `r` 处沿方向 `s` 的辐射强度
- `β = κ + σ_s`：总消光系数
- `κ`：吸收系数
- `σ_s`：散射系数
- `Φ(s,s')`：散射相函数（Henyey-Greenstein）
- `g`：不对称参数（-1到1）

### 1.2 研究挑战

1. **高散射区域** (`σ_s/β ≈ 0.9`)：强散射使得传统离散坐标法（DOM/S_N）计算成本高昂
2. **各向异性散射** (`g = 0.8`)：强前向峰值散射要求精确的角向离散
3. **验证困境**：缺乏针对各向异性三维RTE的高保真基准解
4. **PINN训练不稳定性**：高散射案例的收敛问题

## 2. 方法论

### 2.1 物理信息神经网络架构

**网络配置：**
- 输入：5维 `(x, y, z, θ, φ)` — 空间坐标+方向角
- 输出：1维 `I(x,y,z,θ,φ)` — 辐射强度
- 架构：8-10个隐藏层，每层128-256个神经元
- 激活函数：高散射用Swish，吸收主导用Tanh
- 残差连接：深层网络稳定性

**损失函数：**
```python
L_total = L_boundary + λ·L_residual

L_boundary = MSE(I_pred, I_exact) 在 ∂Ω 上
L_residual = MSE(R[I_pred], 0) 在 Ω 内

其中 R[I] = (s·∇)I + βI - κI_b - σ_s/(4π) ∫ IΦdΩ'
```

**训练策略：**
- 两阶段优化：
  1. Adam预热（2000次迭代，λ=0.1）— 侧重边界
  2. L-BFGS精调（λ=1.0）— 强化PDE约束
- 梯度累积：内存效率优化
- 权重退火：平衡收敛

### 2.2 前向蒙特卡洛求解器

**方法：轨迹长度估计器（Track-Length Estimator）与生存偏置**

核心算法：
```
对每个光子束：
  1. 从源分布采样初始位置 r_0 ~ S(r) = max(0, 1-5r)
  2. 采样各向同性初始方向 s_0
  3. 当光子活跃时：
     a. 采样光程 τ ~ Exp(1)
     b. 计算自由飞行距离 l = τ/β
     c. 以微步长 ds 追踪路径
     d. 累积：G_cell += 权重 × ds / V_cell
     e. 应用生存偏置：权重 *= σ_s/β
     f. 若权重 < 10^-4 则进行俄罗斯轮盘赌
     g. 若幸存，用HG相函数散射
```

**关键特性：**

1. **轨迹长度估计器（TLE）**：
   - 沿飞行路径连续沉积能量
   - 对光学厚介质比碰撞估计器效率更高
   - 微步长：`ds = min(dx,dy,dz)/5`

2. **生存偏置（隐式俘获）**：
   - 避免光子过早终止
   - 权重衰减：`w ← w × ω` 每次碰撞
   - 保持能量守恒

3. **俄罗斯轮盘赌**：
   - 低权重光子的方差缩减
   - 10%生存概率，10倍权重提升
   - 消除"权重退化"

4. **Henyey-Greenstein相函数**：
   ```
   P(cosθ) = (1-g²)/(4π(1+g²-2g·cosθ)^(3/2))
   ```
   - 反演法精确采样
   - 处理 `g = 0.0`（各向同性）到 `g = 0.8`（强前向）

### 2.3 三级验证体系

**Level 1：纯吸收（Case A）**
- 参数：`κ = 5.0, σ_s = 0.0, g = 0.0`
- 验证：Beer-Lambert定律解析解
- 目的：验证源项和衰减

**Level 2：各向同性散射（Case B）**
- 参数：`κ = 0.5, σ_s = 4.5, g = 0.0`（β = 5.0, ω = 0.9）
- 目的：验证各向同性散射积分

**Level 3：各向异性前向散射（Case C）**
- 参数：`κ = 0.1, σ_s = 4.9, g = 0.8`（β = 5.0, ω = 0.98）
- 目的：验证HG相函数实现

### 2.4 基准生成策略

**标准版本（500万光子）**：
- 开发期间快速验证
- 相对标准差：~5%

**高统计版本（2亿光子）**：
- 出版级降噪
- 相对标准差：~1%

**终极版本（10亿光子+后处理）**：
- **暴力计算**：1,000,000,000个光子束
- **三维八象限对称**：平均8个对称象限
  ```python
  G = (G + G[::-1,:,:])/2  # X对称
  G = (G + G[:,::-1,:])/2  # Y对称
  G = (G + G[:,:,::-1])/2  # Z对称
  ```
- **三维高斯扩散**：`σ = 0.6` 去除残余噪声
- 相对标准差：<0.5%

## 3. 实现细节

### 3.1 源项解耦

传统方法：源与吸收耦合 `κ·I_b`

我们的创新：**纯数学源**
```python
S(r) = max(0, 1 - 5r) 当 r < 0.2
```

**理由**：
- 即使κ很小也确保足够能量
- 严格测试散射积分（主导项）
- 与PINN训练设置完全匹配

### 3.2 坐标系对齐

**关键物理约束**：

| 网格类型 | 坐标定义 | X轴 |
|---------|---------|-----|
| FMC (MC) | 单元中心 | `x_i = (i+0.5)/N` |
| PINN (NN) | 节点中心 | `x_i = i/(N-1)` |

**应用的修正**：
```python
# FMC单元中心
x_fmc = np.linspace(0.5/50, 1-0.5/50, 50)

# PINN节点位置
x_pinn = np.linspace(0, 1, n_points)

# 将PINN插值到FMC坐标
G_aligned = np.interp(x_fmc, x_pinn, G_pinn)
```

没有这种对齐，陡峭梯度区域会出现~10%的幻影误差。

### 3.3 中心线提取

**问题**：物理中心 `(0.5, 0.5, 0.5)` 位于单元24和25之间的面上

**解决方案**：平均4个中心单元
```python
iy1, iy2 = 24, 25  # nx//2-1, nx//2
iz1, iz2 = 24, 25
G_centerline = np.mean(G[:, iy1:iy2+1, iz1:iz2+1], axis=(1,2))
```

**对称折叠**（有效统计量翻倍）：
```python
mid = 25
left = G_centerline[:mid]
right = G_centerline[mid:][::-1]
G_symmetric = np.concatenate([(left+right)/2, (left+right)/2[::-1]])
```

## 4. 验证结果

### 4.1 FMC内部验证

**Case A（纯吸收）**：
- 预期：`G_center ≈ 0.92`（Beer-Lambert）
- FMC结果：`G_center = 0.91 ± 0.02` ✓

**Case B（各向同性）**：
- FMC：`G_center = 2.15 ± 0.05`
- 散射使G增加~2.3倍（光子俘获效应）

**Case C（前向）**：
- FMC：`G_center = 1.67 ± 0.04`
- 低于Case B：前向散射减少横向扩散

### 4.2 PINN与FMC对比

**误差指标**：
```
Case B（各向同性）：
  L2相对误差：  X.XX%
  最大绝对误差：X.XXXX
  
Case C（各向异性）：
  L2相对误差：  X.XX%
  最大绝对误差：X.XXXX
```

**物理解释**：
- L2误差 < 5%：极好的一致性
- L2误差 5-10%：良好的一致性（工程可接受）
- L2误差 > 10%：需要改进模型

### 4.3 收敛分析

**PINN训练收敛**：
- Case A：快速收敛（1000轮）
- Case B：中等（3000轮）
- Case C：慢收敛（5000+轮，高散射）

**学习率策略**：
- Case A：激进（lr=0.01）
- Case B/C：保守（lr=0.001）+梯度裁剪
